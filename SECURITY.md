## 🚫 O que não é permitido

Não é permitido, sem autorização prévia e por escrito:

* copiar este projeto total ou parcialmente;
* clonar a ideia para criar um app semelhante;
* reutilizar telas, fluxos, textos, estilos ou identidade visual;
* adaptar o projeto para outro nome/marca;
* usar o código em projetos pessoais, acadêmicos, comerciais ou profissionais;
* vender, redistribuir ou publicar versões derivadas;
* usar o Ritual como base para outro produto;
* reproduzir funcionalidades, layout ou experiência visual de forma substancialmente semelhante;
* utilizar imagens, textos, assets ou componentes deste repositório;
* apresentar este projeto como se fosse de outra pessoa;
* treinar, gerar ou montar projetos derivados a partir deste código ou design.

## 🛡️ Segurança e privacidade

O Ritual lida ou poderá lidar com informações pessoais e sensíveis, como:

* rotina diária;
* hábitos;
* ciclo;
* sintomas;
* humor;
* metas;
* registros corporais;
* alimentação;
* círculos e desafios.

Por isso, o desenvolvimento do app considera:

* proteção de dados;
* autenticação segura;
* regras de acesso;
* isolamento de dados por usuário;
* cuidado com informações sensíveis;
* revisão de permissões;
* boas práticas de segurança.
