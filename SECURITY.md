# Segurança do pacote público

Este diretório é somente para visualização do MVP.

Não publique aqui:

- arquivos .env;
- migrations Supabase;
- Edge Functions;
- scripts internos;
- Android project;
- chaves privadas;
- service role;
- tokens de Firebase;
- documentação de auditoria interna.

Se precisar alterar regras de dados, faça isso no repositório privado do Ritual e publique novamente apenas o build estático.
